# Ovarian Cancer Classification using MLP

## Overview
Binary classification of ovarian cancer using 
a Multilayer Perceptron (MLP) neural network.

## Dataset
ovariantotal.csv

## Results
- Training Accuracy: 79.88%
- Test Accuracy: 84.29%

## Tech Stack
Python | TensorFlow | Keras | Pandas | NumPy | Scikit-learn

## How to Run
1. Open notebook in Google Colab
2. Upload ovariantotal.csv
3. Run all cells
